# CodeQLScanRepo

Starter source for the **CodeQL Security Scan** GitHub Actions lab.

This repository contains a minimal, analyzable JavaScript project. Copy these
files into your own repository, then add the CodeQL workflow at
`.github/workflows/codeql-analysis.yml` as described in the lab instructions.

```
CodeQLScanRepo/
├── package.json
└── src/
    └── app.js
```

> Note: This repo intentionally does **not** include the workflow file —
> creating it is part of the lab task.
